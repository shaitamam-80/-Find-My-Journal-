# Find My Journal - סקירת פרויקט

## מהו Find My Journal?

**Find My Journal** הוא אפליקציית SaaS שעוזרת לחוקרים ואקדמאים למצוא את הכתבי-עת המתאימים ביותר לפרסום המאמרים שלהם.

---

## הבעיה

חוקרים מתמודדים עם אתגר משמעותי: **איך לבחור את הכתב-עת הנכון לפרסום?**

### האתגרים העיקריים:

1. **עומס מידע** - קיימים עשרות אלפי כתבי-עת אקדמיים בעולם
2. **התאמה לא ברורה** - קשה לדעת אילו כתבי-עת באמת מתאימים לנושא המחקר
3. **חיפוש ידני ממושך** - חוקרים מבזבזים שעות רבות בחיפוש ידני
4. **סיכון לדחייה** - הגשה לכתב-עת לא מתאים מובילה לדחייה ובזבוז זמן
5. **כתבי-עת טורפניים** - קושי לזהות כתבי-עת לא אמינים (predatory journals)

---

## הפתרון

Find My Journal מציע **חיפוש חכם** המבוסס על אלגוריתם היברידי המשלב:

### 1. התאמת נושאים (Topic Matching)
- ניתוח אוטומטי של כותרת ותקציר המאמר
- זיהוי הנושאים המרכזיים באמצעות OpenAlex API
- התאמה לכתבי-עת שמפרסמים מאמרים בנושאים דומים

### 2. זיהוי תחום אוטומטי (Discipline Detection)
- זיהוי אוטומטי של התחום האקדמי (רפואה, הנדסה, מדעי החברה וכו')
- תמיכה ב-252 תת-תחומים אקדמיים
- שיפור דיוק ההמלצות בהתאם לתחום הספציפי

### 3. אלגוריתם Soft Boost
- כתבי-עת בתחום המשתמש מקבלים ניקוד מוגבר
- **אבל**: תוצאות עם התאמת נושאים חזקה לא מוסתרות לעולם
- מאפשר גילוי הזדמנויות פרסום בינתחומיות

## יכולות המערכת

### מה המערכת מספקת:

| יכולת | תיאור |
|-------|-------|
| **חיפוש מהיר** | תוצאות תוך שניות |
| **ציון התאמה** | אחוז התאמה (0-100%) לכל כתב-עת |
| **מדדי כתב-עת** | H-Index, מספר פרסומים, סטטוס Open Access |
| **זיהוי APC** | הצגת עלויות פרסום (Article Processing Charges) |
| **סינון כתבי-עת טורפניים** | הגנה מפני כתבי-עת לא אמינים |
| **תמיכה ב-40+ שפות** | כיסוי בינלאומי רחב |
| **מצב פרטיות** | אפשרות חיפוש ללא שמירת היסטוריה |

### מה המערכת **לא** מספקת:

- Impact Factor (דורש מנוי ל-Clarivate)
- שיעורי קבלה (מידע לא זמין ב-API)
- זמני פרסום משוערים (מידע לא זמין)

## ארכיטקטורה טכנית

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│                 │     │                 │     │                 │
│    Frontend     │────▶│    Backend      │────▶│   OpenAlex API  │
│  React + Vite   │     │    FastAPI      │     │  (Data Source)  │
│                 │     │                 │     │                 │
└─────────────────┘     └─────────────────┘     └─────────────────┘
        │                       │
        │                       │
        ▼                       ▼
┌─────────────────┐     ┌─────────────────┐
│                 │     │                 │
│     Vercel      │     │    Supabase     │
│   (Hosting)     │     │  (Auth + DB)    │
│                 │     │                 │
└─────────────────┘     └─────────────────┘
```

### Stack טכנולוגי:

| שכבה | טכנולוגיה |
|------|-----------|
| **Frontend** | React, TypeScript, Vite, Tailwind CSS |
| **Backend** | Python, FastAPI |
| **Database** | PostgreSQL via Supabase |
| **Auth** | Supabase GoTrue (JWT) |
| **Data Source** | OpenAlex API |
| **Deployment** | Vercel (Frontend), Railway (Backend) |

              ▼

## מקור הנתונים: OpenAlex

[OpenAlex](https://openalex.org/) הוא מאגר פתוח וחינמי של מידע אקדמי הכולל:
- מעל 250 מיליון עבודות אקדמיות
- מעל 100,000 כתבי-עת
- מידע על מחברים, מוסדות ונושאים
- עדכונים יומיים

**למה OpenAlex?**
- חינמי לשימוש
- API פתוח ונגיש
- מידע עשיר ומעודכן
- קהילה פעילה
